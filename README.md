Project Overview:-
DeepWind AI is an intelligent system that uses hybrid deep learning models (CNN, RNN, and classical ML algorithms) to predict wind turbine power output and detect operational faults.
The system aims to improve energy efficiency, reduce downtime, and enable predictive maintenance for modern wind energy systems.
